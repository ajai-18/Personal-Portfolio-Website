# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website built using **HTML**, **CSS**, and **JavaScript** to showcase my skills, projects, education, certifications, and contact information.

## 🚀 Features

- Responsive design for desktop, tablet, and mobile devices
- Modern glassmorphism-inspired user interface
- Hero section with profile image
- About Me section
- Education details
- Technical and soft skills
- Project showcase
- Internship experience
- Certifications and achievements
- Contact form with direct email functionality
- Social media links (LinkedIn, GitHub, LeetCode)
- Resume download option

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive Web Design
- Glassmorphism UI

## 🚀 How to Run

1. Clone the repository

```bash

```

2. Open the project folder

```bash
cd portfolio
```

3. Open `index.html` in your browser.

No additional installation is required.
